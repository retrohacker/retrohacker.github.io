Breaking Ground on Persona

Breaking ground on Persona, an Cloudflare Native application built on top of OpenAI. Persona allows you to define your own "personas" that will "translate" text by rewriting it as if it were written by that persona. So you can write your text as you normally would and Persona will rewrite it as if someone else had written it. In this stream, we create the project and make HTTP requests to the OpenAI API.

Topics: Workers, Pages, OpenAI
